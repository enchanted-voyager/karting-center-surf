# Требования к MVP: приложение «Картинг-центр»

## 1. Описание продукта

Мобильное/веб-приложение для картинг-центра, которое позволяет клиентам
знакомиться с трассами и тарифами, а также бронировать заезды онлайн без
необходимости звонить администратору.

## 2. User Stories

1. **Просмотр каталога трасс**
   Я, как клиент, хочу посмотреть список доступных картинг-трасс с их
   описанием, сложностью и фотографиями, чтобы выбрать подходящую трассу
   перед посещением центра.

2. **Бронирование заезда**
   Я, как клиент, хочу забронировать конкретную дату и время заезда онлайн,
   чтобы не тратить время на ожидание и звонки администратору.

3. **Просмотр личных бронирований**
   Я, как клиент, хочу видеть список своих текущих и прошлых бронирований,
   чтобы отслеживать историю посещений и не забыть о предстоящем заезде.

4. **Отмена/перенос бронирования**
   Я, как клиент, хочу иметь возможность отменить или перенести бронь,
   чтобы гибко управлять своими планами в случае изменения обстоятельств.

## 3. Основные сценарии использования

### Сценарий 1: Просмотр каталога трасс

1. Пользователь открывает приложение и переходит в раздел «Трассы».
2. Приложение отображает список трасс с фото, названием, уровнем сложности
   и стоимостью заезда.
3. Пользователь выбирает интересующую трассу.
4. Приложение показывает подробную карточку трассы: описание, схему трассы,
   длину, ограничения (возраст, рост) и доступные тарифы.

### Сценарий 2: Бронирование заезда

1. Пользователь находится на карточке выбранной трассы и нажимает
   «Забронировать».
2. Приложение предлагает выбрать дату и доступное время сеанса из
   расписания.
3. Пользователь указывает количество участников и выбирает тариф.
4. Пользователь подтверждает бронирование.
5. Приложение сохраняет бронь, отображает подтверждение и добавляет запись
   в раздел «Мои бронирования».

## 4. Ключевые сущности данных (для MVP)

- **Трасса** — название, описание, фото, сложность, длина, ограничения.
- **Сеанс/расписание** — дата, время, доступные слоты, тариф.
- **Бронирование** — пользователь, трасса, сеанс, количество участников,
  статус (активно/отменено/завершено).
- **Пользователь** — имя, контактные данные.

## 5. Вне рамок MVP

- Онлайн-оплата.
- Программа лояльности и бонусы.
- Рейтинги и статистика заездов (круги, время).
- Push-уведомления.
